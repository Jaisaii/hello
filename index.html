ABOUT GITHUB

Useful Git Commands Recap

-> Clone a repository: git clone https://github.com/username/repository-name.git


->Check status: git status


->Add changes: git add .


->Commit changes: git commit -m "Your commit message"


->Push changes: git push origin branch-name


->Create a branch: git checkout -b new-branch-name


->Switch branches: git checkout branch-name


->Merge branches: git merge branch-name


->Fetch updates: git fetch


->Pull updates: git pull origin branch-name


->View commit history: git log


-> Commit is used to report changes in repository


-> Repository is a centeralized place to store and manage codes


-> Branch is used to develop features, fix bugs and experiment with new ideas in isolate from the main code.
